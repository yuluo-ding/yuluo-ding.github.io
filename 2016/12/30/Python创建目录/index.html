Python创建目录

Python 2016/12/30

本文于596天之前发表，文中内容可能已经过时。

流程如下：

  1. 判断目录是否存在os.path.exists(path)
  2. 创建多层目录os.path.makedirs(path)
  3. 创建目录os.mkdir(path)

当父目录不存在的时候os.mkdir(path)不会创建，os.makedirs(path)则会创建父目录
