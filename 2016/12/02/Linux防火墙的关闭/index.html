Linux防火墙的关闭

Posted on 2016-12-02 | In Linux

在连接MySQL中，遇到ERROR 2002 (HY000): mysql

这个其实是Linux防火墙的阻拦

解决办法就是关闭防火墙即可。

步骤如下：

  1. 立即关闭：service iptables stop
  2. 永久关闭：iptables -F
  3. 禁止启动：chkconfig iptables off
  4. 另外，运行 setup 在界面，选择Firewall configuration，进入下一界面，选择 Security Level为Disabled，保存。
