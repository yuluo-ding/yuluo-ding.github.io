恢复Chrome的地址栏www

  1. 打开chrome
  2. 地址栏输入
  3. chrome://flags/#omnibox-ui-hide-steady-state-url-scheme-and-subdomains
  4. 将选项改为disabled
  5. 重启浏览器

恢复HTTPS 安全图标

  1. chrome://flags/#simplify-https-indicator
  2. 将选项改为
  3. enabled（show secure chip for non-ev https pages）
  4. 重启浏览器
