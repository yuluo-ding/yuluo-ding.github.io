剩下的其余参数如下：

  • %y 两位数的年份表示
  • %Y 四位数的年份表示
  • %m 月份
  • %d 月内中的一天
  • %H 24小时制小时数
  • %I 12小时制小时数
  • %M 分钟数
  • %S 秒
  • %a 本地简化星期名称
  • %A 本地完整星期名称
  • %b 本地简化的月份名称
  • %B 本地完整的月份名称
  • %c 本地相应的日期表示和时间表示
  • %j 年内的一天
  • %p 本地A.M.或P.M.的等价符
  • %U 一年中的星期数星期天为星期的开始
  • %w 星期，星期天为星期的开始
  • %W 一年中的星期数星期一为星期的开始
  • %x 本地相应的日期表示
  • %X 本地相应的时间表示
  • %Z 当前时区的名称
  • %% %号本身
