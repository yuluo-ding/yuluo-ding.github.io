adb shell 错误

Study Class 2015/05/16

本文于1535天之前发表，文中内容可能已经过时。

使用adb shell提示adb server is out of date. killing...?

#####是ADB端口被占用了

先执行adb nodaemon server ，查看adb server的端口是多少

再执行下netstat -ano | findstr "5037"

最后执行tskill 4236即可。
