让Chrome显示www和https

新版的Chrome 77又把www和https给隐藏了，日常搞事啊，恢复显示的步骤如下

在浏览器的地址栏输入以下两个命令：

  1. chrome://flags/#omnibox-ui-hide-steady-state-url-scheme，然后选择disabled
  2. chrome://flags/#omnibox-ui-hide-steady-state-url-trivial-subdomains，然后选择disabled
  3. 重启Chrome浏览器

重新打开网站，应该就能恢复了。
