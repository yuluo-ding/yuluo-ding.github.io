HTTP协议下的分段下载

在 HTTP 协议下分段下载文件

假如一个文件有1000个字节，那么其范围就是0-999，则：
Range: bytes=500- 表示读取该文件的500-999字节，共500字节。
Range: bytes=500-599 表示读取该文件的500-599字节，共100字节。
