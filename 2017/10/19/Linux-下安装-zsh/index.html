Linux 下安装 zsh

Linux 2017/10/19

本文于403天之前发表，文中内容可能已经过时。

Zsh 是一款功能强大终端（shell）软件，既可以作为一个交互式终端，也可以作为一个脚本解释器。它在兼容 Bash 的同时 (默认不兼容，除非设置成 emulate sh) 还有提供了很多改进。

安装更新过程：

1. 安装 zsh 软件包和 git
2. 克隆 oh-my-zsh
3. 复制 .zshrc，修改命令提示符样式
4. 修改 shell 类型

5. 更新 oh-my-zsh

6. 如果要换 oh-my-fish
